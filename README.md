# LegeSystem

A terminal-based medical prescription management system built in Java, now showcased as a live interactive demo via a Node.js web app.

Originally a university group project combining multiple OOP concepts — patients, doctors, medications, and prescriptions with full CRUD interactions.

## Features

- Manage patients, doctors, medications, and prescriptions
- Write and use prescriptions (white, blue, military, P-type)
- View statistics on narcotic and habit-forming medication usage
- Live interactive terminal demo in the browser via WebSockets and xterm.js
